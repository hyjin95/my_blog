# 단축키

{% hint style="info" %}
그날그날 배우고 업데이트 합니다.

그렇기 때문에 어수선할 수 있습니다.
{% endhint %}

### Eclipse IDE

* system          : "sys" + ctrl + space bar
* 복사               : ctrl + alt + ↓
* 코드 줄 이동 : alt + 방향키
* import 추가  : ctrl + shift + o
* 지우기           : ctrl + d
* 일괄처리       : alt + shift + a
* 들여쓰기       : 드래그 + tab

### 깃북

* 줄 바꾸기 : shift + enter

### Toad for Oracle

* Describe : select + f4
