## 흐름제어 구문

### 흐름제어 구문의 종류

```cpp
//조건문 if
if(조건)
    문장1; //조건이 true면 실행
else
    문장2; //조건이 false면 실행
```

* 조건문
  * 조건에 따른 실행 흐름 제어
  * if문,switch문
* 반복문
  * 정해진 범위의 문장을 반복 실행하는 문장
  * for문, while문, do-while문
